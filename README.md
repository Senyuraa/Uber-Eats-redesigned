This project is a **redefined and reimagined frontend UI** inspired by modern food delivery platforms.

It focuses on **visual hierarchy, spacing, typography, and layout design**, while adding a fresh interpretation instead of directly copying any existing product.

---

## Highlights

- Full-screen hero section with cinematic video background  
- Modern typography and strong visual scale  
- “Better food for more people” data section  
- App features section with a scattered, breathable layout  
- Large black footer inspired by real-world platforms  
- Clean, premium, product-style UI
